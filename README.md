# Z-Trib 

## Présentation
Application pour la gestion d'événements sportif qui 
  - met en relation des sportifs amateurs pour organiser des matchs, des tournois, des entrainements
  - permet pour un club d'organiser des évenements

## Fonctionnalités de l'application
  - Compte d'un sportif
    - Nom / Pseudo
	   - Majeur (Oui / Non)
	   - Region
	   - adresse mail
	   - Sports pratiqués
	   - un avatar
	   - mes stats sur les événements auxquels j'ai participé
 
  - Mes tribus
    - Possibilité de créer des tribus et d'invité de personnes
	       
  - Mes évenements
    - Evenement autonomme : Je créé un événement sans inviter un utilisateur sur un sport précis avec un type de tournoi précis
      - Je gere tout tout seul
	  - J'inscris les équipes
	  - Je rentre les résultats
	  - Je mets à dispo durant le tournoi un QR CODE
	  - Je fournis un réglement
      - Je données les modalités d'accés (plus orienté club)
	  - Je donne ce qui sera dispo sur place (café, sandwich, etc...)
    - Evenement pour ma tribu : Je créé un événement avec les membres de ma tribu sur un sport précis avec un type de tournoi précis
      - les membres sont autonommes pour saisir les résultats
	  - Peuvent proposer des lieux pour avoir un tournoi sur plusieurs semaine en définissant des rencontres
	  - Je fournis un réglement
  - type de tournoi possible
    - Championnat tout le monde rencontre tout le monde
	- Poule en phase 1 puis poule de classement en phase 2
	- Poule en phase 1 puis phase éliminatoire phase 2
	- Match unique
